# User Journeys

## Worker
1. Sign up → verify email (Phase 2)
2. Complete profile
3. Submit KYC → pending → approved/rejected
4. Wallet becomes eligible → virtual card issue
5. Earn via milestones → payments released → wallet credited
6. Spend via virtual card → transactions show in feed

## Client
1. Sign up → company setup
2. Invite worker → contractor onboarding
3. Create milestones/invoices
4. Fund wallet → release payments → receipts

## Admin
1. Review KYC queue
2. Monitor transactions, payouts, jobs
3. Handle disputes and reversals
